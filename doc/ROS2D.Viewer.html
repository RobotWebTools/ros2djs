Class: Viewer

ROS2D. Viewer

new Viewer(options)

A Viewer can be used to render an interactive 2D scene to a HTML5 canvas.
Parameters:
Name Type Description
options object with following keys: * divID - the ID of the div to place the viewer in * width - the initial width, in pixels, of the canvas * height - the initial height, in pixels, of the canvas * background (optional) - the color to render the background, like '#efefef'
